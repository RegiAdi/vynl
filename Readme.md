# Lazy Code(MVC) Generator for Express JS

1. Install packages

npm

```
$ npm install
```

yarn

```
$ yarn install
```

2. Create src, src/models, src/controllers, src/routes in project directory
3. Generate all by running

```
Lazy CLI

lazy [command]

Commands:
  lazy init                             Initializes project
  lazy generate                         Run lazy generator

Options:
  --help      Show help
```
